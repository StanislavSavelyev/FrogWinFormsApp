# FrogWinFormsApp
Игра-головоломка «Лягушки» на Windows Forms. Нужно поменять лягушек местами, перепрыгивая через пустую клетку.

## Правила игры

- На игровом поле 8 лягушек: 4 слева (зелёные) и 4 справа (коричневые)
- Одна пустая клетка посередине
- Лягушка может переместиться на пустую клетку, если она находится рядом (на расстоянии 1 клетки)
- **Цель:** собрать всех «правых» (коричневых) лягушек в левой зоне (слева от пустой клетки)

## Технические особенности

- **Платформа:** .NET 10.0, Windows Forms
- **Графика:** PictureBox с изображениями лягушек из ресурсов приложения
- **Логика:**
  - Проверка допустимости хода (расстояние между кликнутой и пустой клеткой не более 2)
  - Обмен позициями (Swap) между кликнутой и пустой клеткой
  - Подсчёт количества ходов
  - Проверка победы (все правые лягушки слева от пустой клетки)
- **Ресурсы:** изображения хранятся в `.resx` (frog1, frog2, frog3)

## Управление

Кликните мышью на любую лягушку — если она может перепрыгнуть на пустую клетку, она переместится.

## Цель игры

Добиться расположения, при котором:
- Слева от пустой клетки находятся **все 4 коричневые лягушки**
- Счёт ходов минимален (идеально — 24 хода)

## Технологии

- C# / .NET 10.0
- Windows Forms
- System.Drawing (работа с изображениями)
